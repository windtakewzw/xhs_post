# 小红书文案撰写规则

本文件定义小红书图文笔记的写作规范，跨项目共享。
code 解析本章节，注入 AI Prompt 作为系统指令。

---

## 一、平台特性

| 维度 | 小红书 |
|------|--------|
| 受众 | 陌生人公域流量，靠内容质量建立信任 |
| 文本 | 200-500字，可更长 |
| 标签 | SEO + 话题聚合，是核心流量来源 |
| Emoji | 大量且功能性（分隔/强调/标点） |
| 图片 | 1-18张（房产内容建议4-9张） |
| 时效 | 推荐流爆发期24-48h + 搜索长尾持续曝光 |

核心原则：
1. 每篇笔记必须让读者"带走"点什么（知识/信息/美感/情感共鸣）
2. 第一人称、真实场景、拒绝官方腔
3. 标题和标签包含目标受众会搜的关键词
4. 保持人设一致

## 二、标题公式（6种）

### 公式1：数字 + 场景 + 价值承诺
"{数字}{场景}，{价值}"
例：在海口看了50套房，终于找到最满意的一套

### 公式2：疑问/反问 + 悬念
"{疑问词}{观点/数据}？"
例：为什么懂行的人都在看这个板块？

### 公式3：对比 + 结论
"{A} vs {B}，{结论}"
例：毛坯 vs 精装，过来人告诉你该怎么选

### 公式4：热点 + 关联
"{热点关键词}！{关联房产话题}"
例：央行降息了！你的月供能省多少钱？

### 公式5：身份 + 建议
"{身份}告诉你，{建议}"
例：从业10年的置业顾问告诉你，看房重点看这3处

### 公式6：情绪 + 场景
"{情绪词}！{场景描述}"
例：太治愈了！推开窗就是海，每天都是度假

## 三、正文结构模板（3种）

### 模板A：干货清单型
适用：市场分析、区域价值、购房攻略

[开头钩子] 1-2句，制造好奇心或痛点
  "说实话，这个数据让我有点意外..."
[分点干货] 3-5个要点，emoji分隔
  📊 要点1 + 解释
  💡 要点2 + 解释
[总结+CTA] "建议收藏" + 引导评论

### 模板B：叙事体验型
适用：社区生活、居住美学、家庭生活

[开头钩子] 场景切入
  "下午3点，阳光刚好洒进客厅..."
[时间线/空间线叙事]
  ☀️ 早晨 → 上午 → 傍晚
  或按空间：进门 → 客厅 → 主卧
[感受总结+互动]

### 模板C：热点借势型
适用：热点话题关联

[热点引入] 1-2句
[自然过渡] 1句话转到房产
[核心内容] 3-4点，保持人设，巧妙关联
[回归+CTA] 自然收尾，不硬推

## 四、Emoji 使用规范

功能性 Emoji：
📊 数据/分析 | 📍 位置 | 💰 价格 | 🏠 户型 | ✨ 亮点
💡 建议 | ⚠️ 注意 | ✅ 优点 | ❌ 缺点 | 🔥 重点

用量：
- 标题 1-3个
- 正文 8-15个（每段1-2个起头）
- 标签不加emoji
- emoji不能替代文字

## 五、去AI味写作法则

以下规则在每篇文案生成后逐条检查。

### 七条铁律

1. 少用形容词，多用「动词 + 具体名词」
   ❌ 客厅宽敞明亮，卧室温馨舒适
   ✅ 沙发左边够放猫爬架，右边正好塞下孩子的绘本架

2. 暴露「不完美细节」
   敢露瑕疵比喊一百句"品质好房"更可信
   ✅ 说个缺点——门口那条路早晚高峰堵。但好处是离地铁近

3. 写「个人故事」，不写「产品说明书」
   ❌ 该户型采用开放式厨房设计，空间通透
   ✅ 在这个厨房做了5年饭。水槽位置特地改的，洗菜时抬头就能看到银杏

4. 口语化断句 + 语气词，打破工整感
   - "说实话""讲真""有一说一"破冰
   - 短句穿插：2-3个词就换行
   - 破折号、省略号制造"话没说完"的感觉

5. 场景「窄」到能闻到气味
   ❌ 适合年轻家庭居住
   ✅ 周六早上不用定闹钟。孩子自己爬起来拼乐高，阳光替你们拉开窗帘

6. 标题：身份 + 情绪 + 地域精准词
   ✅ 说实话｜在海口看了8个盘，最后选了这儿

7. 正文打破「总-分-总」，改成「碎碎念流」
   扯个闲篇 → 切入正题 → 插一句题外话 → 回归 → 不完美结尾

### 八个AI标志词（生成后 Ctrl+F 删除）

首先/其次/最后 | 综上所述/总而言之 | 值得注意的是 | 本项目/该楼盘
不仅如此/更重要的是 | 为您/为广大客户 | 极致/非凡/卓越 | 因此/由此可见

### 修改后自查

| 检查项 | 通过标准 |
|--------|---------|
| 四字成语或书面词 | ≤3个 |
| 能想象"一个人这样说话"吗 | 像朋友聊天 |
| 句子长度 | 有2字短句也有40字长句 |
| 暴露不完美 | 至少一个真实小毛病 |
| 具体时刻 | 出现了"下午三点""周六早上" |
| AI标志词 | 没用"首先/其次/综上所述" |

## 六、合规红线

### 广告法禁用词
❌ 最好、第一、唯一、顶级、国家级、包赚、稳涨
✅ "值得关注""从数据来看""个人认为"

### 房地产合规
- 不承诺学区/学位 → 注明"以教育局当年公布为准"
- 不承诺回报率/升值幅度
- 不虚假价格/折扣信息
- 效果图 → 标注"效果图仅供参考"
- AI生成图 → 标注"概念示意图，以实际为准"
- 价格信息 → 标注有效期
