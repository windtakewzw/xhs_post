# 标签策略规则

定义小红书笔记的话题标签分层策略和模板。跨项目共享。

---

## 一、标签分层策略

```
第一层（1-2个）：大流量标签 → 曝光
第二层（2-3个）：精准城市/领域标签 → 目标受众
第三层（2-3个）：长尾标签 → 精准搜索
总计：5-8个标签，空格分隔
```

## 三、按人设的基础标签

### investment-advisor（投资置业顾问）
基础包：`#置业顾问 #买房攻略 #楼盘分析`

按内容类型追加：
- market-analysis → #楼市走势 #房产政策 #房价分析
- area-value → #区域价值 #城市规划 #板块分析
- product-analysis → #户型推荐 #性价比 #楼盘测评
- buying-guide → #购房避坑 #买房干货 #贷款买房
- trend-jacking → 热点标签 + #房产解读

### lifestyle-advisor（生活品质顾问）
基础包：`#置业顾问 #品质生活 #理想家`

按内容类型追加：
- community-life → #社区生活 #园林景观 #配套
- home-aesthetics → #家居美学 #空间设计 #家的样子
- product-analysis → #看房日记 #实景拍摄 #户型
- family-living → #品质生活 #生活方式 #社区
- trend-jacking → 热点标签 + #生活共鸣

### family-advisor（家庭置业顾问）
基础包：`#置业顾问 #家庭买房 #买房干货`

按内容类型追加：

## 五、热点标签规则

热点标签来源：抖音热榜、微博热搜、小红书趋势

使用规则：
1. 仅在与内容相关时使用
2. 不超过标签总数的 1/3（最多2个）
3. 热点过后，笔记仍保有长尾标签的搜索价值
4. 负面热点不碰（房企暴雷、维权、灾害等）

## 六、标签生成检查清单

生成标签后检查：
- [ ] 总数 5-8 个
- [ ] 包含城市名
- [ ] 包含项目名（如有）
- [ ] 包含内容类型关键词
- [ ] 包含人设标识
- [ ] 无绝对化用语（"第一""最好"）
- [ ] 热点标签不超过 2 个
- [ ] 标签间空格分隔
