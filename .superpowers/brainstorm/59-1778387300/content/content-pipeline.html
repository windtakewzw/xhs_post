① 素材采集

两种来源混合
官方素材（结构化导入）
  • 开发商提供的户型图、效果图
  • 楼盘基础信息：面积/价格/配套
  • 政策文件、促销活动文案
  • 导入方式：Web上传 / API接入 / 文件夹监控
网络采集（补充素材）
  • 房产平台数据抓取（房天下/贝壳等）
  • 周边配套信息（学校/商圈/交通）
  • 区域内竞品楼盘动态
  • 本地房产政策新闻

②-④ 内容选题 + AI生成 + 差异化

内容类型与生成策略
内容类型 选题来源 AI Prompt 要点 差异化策略
楼盘介绍 官方素材 突出卖点，口语化表达，避免官方腔 不同角度：户型/环境/价格/配套
政策解读 官方政策文件 通俗解读，突出对购房者的利好 不同切入点：利率/首付/资格
客户引流 楼盘活动/节点 制造紧迫感，引导私信/预约看房 不同CTA：看房团/优惠/限时折扣
市场资讯 网络采集 区域房价走势、购房攻略，树立专业形象 不同区域/人群定位
实景看房 官方图片素材 图文搭配，场景化描述 不同户型/楼层/朝向

内容数据模型

SQLite — posts 表
CREATE TABLE posts (
    id              TEXT PRIMARY KEY,
    account_id      TEXT NOT NULL,              -- 发布账号
    project_id      TEXT NOT NULL,              -- 所属楼盘
    content_type    TEXT NOT NULL,              -- 楼盘介绍/政策解读/客户引流/市场资讯/实景看房
    title           TEXT NOT NULL,
    body            TEXT NOT NULL,
    hashtags        TEXT,                        -- JSON: ["#楼盘名", "#城市买房", ...]
    images          TEXT,                        -- JSON: ["local/path/img1.jpg", ...]
    ai_prompt_used  TEXT,                        -- 生成时使用的prompt（调试用）
    status          TEXT DEFAULT 'draft',       -- draft/review/queued/publishing/published/failed
    scheduled_at    TEXT,                        -- 计划发布时间
    published_at    TEXT,
    xhs_note_id     TEXT,                        -- 发布后的小红书笔记ID
    error_msg       TEXT,
    created_at      TEXT DEFAULT (datetime('now'))
);
