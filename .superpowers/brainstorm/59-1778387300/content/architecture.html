关键技术选型建议

层 推荐 理由
语言 TypeScript / Node.js Playwright 原生支持，异步IO适合并发发布
浏览器自动化 Playwright 已安装插件，多BrowserContext隔离账号
数据库 SQLite (开发) / PostgreSQL (生产) 轻量起步，后期可切换
任务队列 BullMQ + Redis 支持定时、重试、优先级、并发控制
管理界面 Web Dashboard (React) 账号管理、内容审核、发布日历可视化
AI 能力 Claude API / 通义千问 文案生成、标题优化、话题推荐
