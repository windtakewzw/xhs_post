xhs_post 完整架构

四大模块
模块 职责 关键实现
1. 素材管理 读取 D:\project\yj_skills\data\ 目录下的项目资料、人设定义、账号配置、营销日历、图片素材 Markdown + YAML frontmatter 解析
素材目录扫描 & 索引
2. 内容生产 选题决策 → AI 生成文案 → 图片生成/匹配 → 去重 → 排期入库
参照 yj_skills references 的设计体系 		营销日历驱动选题

Seedream API 图片生成
内容去重 & 差异化
3. 发布引擎 Playwright 浏览器自动化
多账号调度 & 并发控制
登录态维护 & 风控 		独立 BrowserContext 隔离
移动端模拟 + 反检测
定时调度 + 失败重试
4. 管理界面 Web Dashboard
账号管理 · 内容审核 · 发布日历 · 数据看板 		Flask/FastAPI 后端
React 前端（可选简化为纯后端 + CLI）

目录结构

项目结构规划
xhs_post/
├── src/
│   ├── content/           # 内容生产模块
│   │   ├── planner.py      # 选题决策
│   │   ├── generator.py    # AI文案生成
│   │   ├── imager.py       # 图片生成
│   │   └── templates/      # Prompt模板
│   ├── publisher/         # 发布引擎
│   │   ├── engine.py       # Playwright自动化
│   │   ├── scheduler.py    # 定时调度
│   │   └── anti_detect.py  # 反检测 & 行为模拟
│   ├── accounts/          # 账号管理
│   │   ├── manager.py      # 账号CRUD + 登录
│   │   └── browser.py      # BrowserContext工厂
│   ├── materials/         # 素材加载
│   │   ├── loader.py       # Markdown/YAML解析
│   │   └── indexer.py      # 素材索引 & 检索
│   ├── db/                # 数据层
│   │   ├── models.py       # SQLite ORM
│   │   └── migrations/     # 数据库迁移
│   ├── web/               # Web管理界面
│   │   ├── app.py          # FastAPI应用
│   │   └── routes/         # API路由
│   └── cli.py             # 命令行入口
├── data/                # 运行时数据
│   ├── accounts/           # 每个账号的 userDataDir
│   └── xhs_post.db         # SQLite
├── config/
│   └── settings.yaml       # 全局配置 + API keys
└── requirements.txt
